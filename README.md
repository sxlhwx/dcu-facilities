정보 [
    1. 시설 {
        교내 식당, 편의점, 카페를 비롯한 시설명 (기본 제공)
        시설이 위치한 건물 번호, 건물 이름, 층 (기본 제공)
    }
    2. 건물 {
        교내 건물 번호, 다양한 명칭(공식, 비공식, 한글, 영문) (상세페이지 제공)
    }
    3. 시간 {
        학기/방학, 주중/주말, 평일/공휴일 모든 경우에 대한 운영시간 (상세페이지 제공)
        학교가 지정한 학기/방학 기간, 공휴일 (현재 제공되지 않음)
        현재 날짜, 요일, 시간 (기본 제공)
        현재가 학기/방학, 주중/주말, 평일/공휴일 중 언제에 해당하는지 (현재 제공되지 않음)
    }
]
기능 [
    0. 요약 {
        초록=운영, 노랑=운영 또는 폐쇄까지 한 시간 미만 남음, 회색=폐쇄
        검색. 입력된 알파벳 또는 알파벳+숫자 조합이 건물 번호로 감지되면 건물 번호 검색결과만 제공
        카테고리 탭. 기본적으로 전체 시설 리스트 제공. 5개의 카테고리 중 하나를 선택하면 해당 카테고리의 시설만, 선택된 카테고리를 다시 누르면 전체 시설로 복귀
        데스크탑/모바일 뷰. 사용자의 화면 가로폭에 따라 자동 전환. 우측 하단 버튼으로 직접 설정하면 자동 전환은 꺼짐.
        (구현중)세부 사항 페이지. 검색에 포함되는 태그, 리스트에 표기되지 않는 시설/건물 명칭, 자세한 운영시간, 운영시간 정보 출처에 대한 이미지, 링크와 업데이트 시점.
        (미구현)오류 또는 잘못된 정보 신고, 정보 제공, 도움 제공, 개선점 전달 등을 위한 링크

    }
    1. 운영여부 {
        개방(초록), 개방 또는 폐쇄 임박/기준은 1시간 미만(노랑), 폐쇄(회색)
    }
    2. 시간 {
        개방 또는 폐쇄까지 남은 시간 계산
        다음 개방 또는 폐쇄 시간 하이라이트
        현재 날짜의 속성에 대응하는 운영시간 정보 표시
    }
    3. 검색 {
        카테고리 - 각 카테고리 확인 가능. 선택된 카테고리를 다시 누르면 전체 보기로 돌아감
        검색창에 a~e 까지의 한 자리 알파벳이 입력되어 있는 경우 해당 건물 번호만 노출
        검색창에 a~e 까지의 한 자리 알파벳 + 한 자리 또는 두 자리 숫자가 입력되어 있는 경우 해당 건물 번호만 노출
        이외의 경우 검색창에서는 태그, 모든 시설 명칭, 모든 건물 명칭만 검색됨
    }
    4. 편의 {
        페이지를 실행하는 화면의 가로폭에 따라 데스크탑, 모바일 뷰가 자동 전환
        모바일 뷰에서 일정 수준 이하의 가로폭으로 줄어들 경우 정보 겹침을 방지하기 위해 간략한 표시로 자동 전환
        페이지 최하단 데스크탑/모바일 버튼을 직접 선택해 뷰를 변경할 경우 데스크탑/모바일 뷰 자동전환은 해제
    }
    5. 신뢰성 {
        각 시설 세부 페이지에는 시설 운영 정보의 출처가 되는 이미지와 링크, 업데이트 일자가 제공될 예정 (현재 미완성)
    }
]
